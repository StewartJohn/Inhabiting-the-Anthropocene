Hover over a node to:

  • show title of the corresponding post, or name of corresponding theme, on the graph
  • read a preview of the post, or description of the theme, in this space
  • hovered node has yellow border

Click a node to:

  • show the post, or description of the theme, in the space to the right
  • show author's bio, in this space (click author's name in post to restore if needed)
  • zoom graph to that node and others linked to it
    • clicking theme node show posts that address that theme
    • clicking post node shows themes that post addresses
    • other nodes and edges become faded
    • clicking background zooms back out
  • clicked nodes have green borders.

Hover over an edge to show outlook the post takes on the theme--on graph, and in this space--as explained in description of the theme.
